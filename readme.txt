== Description ==

This Plugin adds a scroll to top button in your site. Please, visit us for more plugins http://www.phpclicks.com/wordpress-scroll-to-top-plugin/

Avoid to mouse scrolling.

Features:

*   You can choose different scroll to top button, setup the background color, size, text color and border radius for button. 
*   Easy to use.
*   Powerfull admin options
*   You can also add a mouse hover text for your button. (New)

== Installation ==

1. Upload `tipu-scroll-to-top.zip` to the `/wp-content/plugins/` directory
2. Unzip archive
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to the Settings Page to update settings if necessary
